5.1-6 ReconstructGroup
‣ ReconstructGroup( C, HomC, ForgetfulFunctor, GeneratingSet, EndImplementation )( function )

Returns: a group

The input is a CAP category C which is equivalent to the skeletal category of finite right \(G\)-sets for some group \(G\), a function HomC computing homs in C (e.g. HomSkeletalFinGSets), a generating set of C, and a function computing ends (e.g. EndAsEqualizer or EndByLifts). The output is a group isomorphic to \(G\).
